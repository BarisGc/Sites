# bduibuttons

## Install

```bash
npm install --save bduibuttons
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'bduibuttons'
import 'bduibuttons/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
